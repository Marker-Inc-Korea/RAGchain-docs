# Utils

## Overview

Utils is set of useful tools in RAGchain. It includes useful tool for obtain better answer results or better retrieval quality. And there are embeddings, slim vector store. Also, we have Linker, the essential part of RAGchain to work well. Or, optional but useful tool like File Cache.
